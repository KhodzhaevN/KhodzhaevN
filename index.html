Добро пожаловать в документацию MyCalculator¶

Введение¶

Это документация для MyCalculator, простого консольного калькулятора.

Установка¶

Чтобы установить MyCalculator, выполните следующую команду:

pip install mycalculator

Использование¶

Для использования MyCalculator импортируйте модуль и выполняйте вычисления:

from mycalculator import add, subtract, multiply, divide

result_add = add(5, 3)
result_subtract = subtract(10, 4)
result_multiply = multiply(2, 6)
result_divide = divide(8, 2)

print("Сложение:", result_add)
print("Вычитание:", result_subtract)
print("Умножение:", result_multiply)
print("Деление:", result_divide)
